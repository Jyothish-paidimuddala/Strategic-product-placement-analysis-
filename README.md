# Strategic Product Placement Analysis Using Tableau

## Project Overview
This project analyzes retail sales data using Tableau to understand the impact of product placement on sales performance. Interactive dashboards help visualize revenue, product categories, regional sales, customer demographics, and discounts to support better business decisions.

## Objectives
- Analyze retail sales data
- Study product placement impact
- Create interactive Tableau dashboards
- Generate business insights and recommendations

## Tools Used
- Tableau Desktop / Tableau Public
- Microsoft Excel
- GitHub

## Dataset
- 1000 Retail Records
- CSV & Excel Format
- 14 Attributes

## Dashboards
- Sales by Shelf Position
- Revenue by Category
- Regional Sales Analysis
- Monthly Revenue Trend
- Top Selling Products
- Discount vs Revenue
- Customer Age Distribution
- Interactive Dashboard

## Key Insights
- Eye-level shelves improve sales.
- Dairy and Snacks generate high revenue.
- Moderate discounts increase sales.
- South region performs best.
- Customers aged 25–40 purchase the most.
