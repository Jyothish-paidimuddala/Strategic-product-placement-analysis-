# Strategic Product Placement Analysis using Tableau

## Project Overview

This project analyzes how product placement affects retail sales and consumer purchasing behavior. Using Tableau, interactive dashboards are created to visualize sales performance, revenue trends, customer demographics, and regional performance. The findings help businesses optimize shelf placement and improve marketing strategies.

## Objectives

- Analyze product placement strategies.
- Identify high-performing product categories.
- Study customer purchasing behavior.
- Compare sales across different regions.
- Provide business recommendations.

## Tools Used

- Tableau Public/Desktop
- Microsoft Excel
- CSV Dataset
- GitHub

## Dashboard Features

- Sales by Shelf Position
- Revenue by Category
- Monthly Sales Trend
- Regional Performance
- Customer Age Analysis
- Top Selling Products
- Discount vs Revenue Analysis
